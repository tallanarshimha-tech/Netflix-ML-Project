# Netflix Supervised Learning Project

## Project Overview

This project applies supervised machine learning techniques on the
Netflix Shows dataset to analyze and predict various content attributes
such as type, release year, duration, rating, and country.

## Dataset

Name: Netflix Shows Dataset\
Source: https://www.kaggle.com/datasets/shivamb/netflix-shows\
Rows: \~8800\
Columns: 12

## Objectives

1.  Predict whether content is a Movie or TV Show\
2.  Predict the release year of content\
3.  Analyze and predict duration\
4.  Classify content as recent or old\
5.  Predict whether content is adult-rated (TV-MA)\
6.  Predict country of production\
7.  Predict year added (popularity proxy)

## Technologies Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## How to Run

1.  Open Kaggle Notebook\
2.  Attach dataset path: /kaggle/input/netflix-shows/netflix_titles.csv\
3.  Run all notebooks
